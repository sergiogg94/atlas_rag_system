RETURN & EXCHANGE POLICY - Version 2.1
Effective: October 2023
Last Updated: January 2024

OUR PROMISE:
We want you to love your shoes. If they're not perfect, we make returns easy and free.

RETURN WINDOW:
- Standard: 60 days from delivery date (extended from 45 days)
- VIP members: 90 days
- Holiday purchases (Nov-Dec): Returns accepted until January 31st

RETURN CONDITIONS:
✓ Shoes must be unworn or lightly worn (we get it - you need to test them!)
✓ Original packaging preferred but not required
✓ No receipt needed (we can look up your order with email)

HOW TO RETURN:

Option 1 - Free Mail Returns:
1. Log into your account > Orders > Select item > "Start Return"
2. Print prepaid return label (FREE - we cover shipping)
3. Drop off at any USPS, UPS, or FedEx location
4. Track your return online

Option 2 - Partner Store Drop-off:
Return at any of our 50+ partner retail locations nationwide (find nearest location on website). Instant store credit available.

REFUND OPTIONS:
- Original payment method: 3-5 business days after we receive return
- Instant store credit: Choose this option for immediate shopping credit + 5% bonus
- Exchange: Priority processing - new item ships same day we receive return

EXCHANGES:
Wrong size? Wrong color? We'll cover shipping both ways for exchanges. New item ships before you return the original (we trust you!).

NO RESTOCKING FEES. EVER.

EXCLUSIONS (Very few):
- Items marked "Final Sale" at checkout (you'll see a warning)
- Shoes worn outdoors extensively (mud, significant scuffing, etc.)
- Items damaged beyond normal try-on wear

DEFECTIVE ITEMS:
Manufacturing defect? We'll replace or refund immediately - no need to return first. Contact customer service with photos. We stand behind quality 100%.

GIFT RETURNS:
Gift recipient can return for store credit without original receipt. Just need order number or packing slip.

QUESTIONS?
Chat with us 24/7 or email returns@athleticfootwear.com
Average response time: Under 2 hours

Changes from v1.8: Extended window to 60 days, free return shipping, removed restocking fees, added partner drop-off locations, instant store credit option, exchange improvements.
